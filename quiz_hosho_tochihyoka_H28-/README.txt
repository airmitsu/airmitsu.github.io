【内容】
このフォルダは GitHub Pages 配置用の問題集PWAアプリ初版です。

【配置】
この中身をそのままリポジトリ直下へ置いてください。

【問題追加】
1. toi#.txt を作成
2. 必要なら kai#.txt を作成
3. 必要なら toi#.(jpg/jpeg/png) を配置
4. problems-index.json に相対パスを追記

【注意】
- GitHub Pages では反映に少し時間がかかることがあります。
- PWAの更新時は、service worker のキャッシュが残ることがあります。
- アイコンは未設定です。必要なら manifest に追加してください。

【今回の実装範囲】
- ホーム画面
- 疑似試験モード
- マスターモード
- 文字サイズ設定
- ローカル保存
- オフライン動作
- 画像ズーム/パン
